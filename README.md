# Matlab2023Project

本项目包含多个基于 MATLAB 的机器人仿真和控制项目。

## 项目结构

### 📁 3DOF-Robot
三自由度机器人相关代码和仿真文件。

### 📁 DOF6-Robot
六自由度机器人相关代码和仿真文件。

### 📁 XB4-Robot
XB4 机器人相关代码和仿真文件。

### 📁 ProjectForCarPole
倒立摆（CartPole）项目相关代码。

### 📁 ProjectForCarPole_RL
倒立摆强化学习（Reinforcement Learning）项目代码。

### 📁 Project_CarPole_MatlabModel
倒立摆的 MATLAB 模型文件。

### 📁 Project_CartPole_MatlabModel_RL
倒立摆强化学习的 MATLAB 模型文件。

## 说明

本项目主要涉及：
- 多自由度机器人建模与仿真
- 倒立摆控制系统
- 强化学习算法应用

## 环境要求

- MATLAB 2023 或更高版本
